PROJECT 1 & 2 


----------- Project 1 File Details -----------

HousePrice.csv: 
The original raw data file that contains the house details along with some missing values and data errors used for testing.

Project1.ipynb: 
The Jupyter Notebook where the data was analyzed, tested, and prototyped step-by-step.

project1.py: 
The main Python script that combines all the cleaning, feature engineering, and validation steps into a clean, automated process.

house_price_production_ready.csv: 
The final, clean data file that is automatically created after the script runs successfully without errors.

README.md: 
The project documentation file that explains how the code worksa and the folder structure.



----------- Project 2 File Details -----------

creditcard.csv:
The original raw dataset containing 284,807 anonymized credit card transactions (PCA-transformed features V1–V28, Time, Amount, and Class) with a highly imbalanced fraud/legitimate ratio, used for training and testing.

Fraud_Detection_Project2.ipynb:
The Jupyter Notebook where the data was explored, split, resampled using SMOTE, and used to train and evaluate multiple classification models (Logistic Regression and Random Forest) step-by-step.

project2.py:
The main Python script that combines all the preprocessing, SMOTE resampling, pipeline construction, model training, and evaluation steps into a clean, automated process.

fraud_detection_results.csv:
The final output file containing model predictions and evaluation metrics (Precision, Recall, ROC-AUC) generated automatically after the script runs successfully without errors.

README.md:
The project documentation file that explains how the code works and the folder structure.
