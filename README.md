----------- Project 1 File Details -----------

HousePrice.csv: 
The original raw data file that contains the house details along with some missing values and data errors used for testing.

Project1.ipynb: 
The Jupyter Notebook where the data was analyzed, tested, and prototyped step-by-step.

project1.py: 
The main Python script that combines all the cleaning, feature engineering, and validation steps into a clean, automated process.

house_price_production_ready.csv: 
The final, clean data file that is automatically created after the script runs successfully without errors.

README.md: 
The project documentation file that explains how the code works, the folder structure, and how to run the script.
